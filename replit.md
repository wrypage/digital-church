# My Python App

## Overview
A Python web application built with Flask. Serves on port 5000.

## Project Structure
- `main.py` — Main application entry point with Flask routes
- `templates/` — HTML templates (Jinja2)
- `static/` — CSS, JS, and other static assets

## Running
```
python main.py
```
The app runs on `0.0.0.0:5000`.
